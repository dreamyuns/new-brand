화면 주석 — MY05-A (이메일 인증)

진입
MY05 회원탈퇴 화면 진입 경로
  • 계정관리(MY03) 하단 [회원탈퇴] 버튼을 탭하면 이 화면으로 이동한다.
  • 로그인한 회원만 진입할 수 있다. 비로그인 상태에서는 이 화면에 접근할 수 없다.
정책서 10-4절 (회원탈퇴 화면 진입).
1
[← 뒤로가기] 버튼
  • 탭 시 이전 화면(MY03 계정관리)으로 돌아간다.
  • 탈퇴가 완료되기 전까지는 언제든지 뒤로 돌아갈 수 있다.
정책서 10-4절 (GNB 뒤로가기만 표시).
2
화면 제목 "회원탈퇴"
  • GNB 가운데에 고정으로 표시된다.
  • [로그인] 버튼과 [☰ 전체메뉴] 버튼은 이 화면에서 나타나지 않는다. (MY01~MY05 마이페이지 계열 공통)
정책서 10-1절 (마이페이지 하위 화면 GNB 패턴).
3
탈퇴 전 안내 문구 및 경고
  • 탈퇴하면 적립된 캐시백이 전액 사라진다는 점을 빨간 박스로 강조 표시한다.
  • "Waiting(적립예정)" 상태와 "Approved(지급완료)" 상태 구분 없이 모두 소멸된다.
  • 탈퇴 후 7일간 같은 이메일로 재가입이 제한됨을 안내한다.
정책서 8-11절 (캐시백 소멸 + 재가입 제한 안내).
◼ 캐시백 잔액 있는 회원: 경고 박스 표시 (현재 화면) ◼ 잔액 없는 회원: 동일 화면 표시 (소멸 금액 0원)
4
가입 이메일 표시 (읽기 전용)
  • 현재 로그인 중인 계정의 이메일 주소가 자동으로 채워진다.
  • 변경할 수 없다 (읽기 전용).
  • 이 이메일 주소로 인증 코드가 발송된다.
정책서 10-4절 (이메일 읽기전용 표시).
5
[인증코드 받기] 버튼
  • 탭 시 위 이메일 주소로 6자리 인증 코드가 발송된다.
  • 발송 후 버튼이 회색으로 바뀌고, 5분 카운트다운이 시작된다.
  • 5분이 지나면 [인증코드 다시 받기] 버튼으로 새 코드를 받을 수 있다.
정책서 8-2절 / 10-4절 (이메일 인증코드 발송, 동일 컴포넌트).
◼ 코드 발송 전: 인디고 버튼 활성 ◼ 발송 후: 회색 비활성, 타이머 시작
6
6자리 인증 코드 입력 박스
  • 숫자 1자리씩 입력할 수 있는 박스 6개가 나란히 있다.
  • 현재 입력 중인 박스에 파란 테두리가 표시된다.
  • 6자리를 모두 입력하는 순간 자동으로 코드 확인이 실행된다. 별도 버튼을 탭하지 않아도 된다.
  • 코드가 올바르면 다음 단계(2단계: 탈퇴 사유)로 자동 이동한다.
  • 코드가 틀리면 입력칸이 빨간색으로 표시되고 "인증 코드가 올바르지 않습니다" 메시지가 나타난다.
정책서 8-2절 (6자리 자동 확인 — 동일 컴포넌트).
7
남은 유효 시간 표시
  • 인증 코드의 유효 시간(5분)이 얼마나 남았는지 카운트다운으로 표시된다.
  • 0:00이 되면 코드가 만료되고, 새 코드를 받아야 한다.
정책서 8-2절 (인증코드 유효시간 5분).
8
[인증코드 다시 받기] 버튼
  • 탭 시 새 인증 코드가 발송되고 타이머가 5분으로 초기화된다.
  • 재전송 횟수 제한은 없다.
정책서 8-2절 (재전송 횟수 제한 없음).
9
[다음 단계로] 버튼 (하단 고정)
  • 6자리 코드가 올바르게 입력될 때까지 회색 비활성 상태로 유지된다.
  • 인증이 완료되면 자동으로 다음 단계(탈퇴 사유 선택)로 넘어간다. 이 버튼을 별도로 탭할 필요 없다.
정책서 10-4절 (인증 완료 후 자동 다음 단계 이동).
◼ 인증 전: 회색 비활성 ◼ 인증 완료: 자동으로 MY05-B로 전환

화면 주석 — MY05-B (탈퇴 사유 선택)

1
[← 뒤로가기] 버튼
  • 탭 시 이전 단계(이메일 인증 화면)로 돌아간다.
  • 이미 선택한 사유는 초기화된다.
정책서 10-4절.
2
탈퇴 사유 선택 안내
  • 아래 6개 항목 중 1개 이상 선택해야 [탈퇴하기] 버튼이 활성화된다.
  • 여러 개를 동시에 선택할 수 있다.
  • 선택한 사유는 서비스 개선을 위한 내부 데이터로만 사용된다.
정책서 8-11절 (탈퇴 사유 6개 + 복수 선택).
3
탈퇴 사유 선택 항목 (1~5번)
  • 항목을 탭하면 동그라미에 체크 표시가 나타나고 파란 테두리가 생긴다.
  • 다시 탭하면 선택이 해제된다.
  • 선택된 항목이 1개 이상이면 하단 [탈퇴하기] 버튼이 빨간색으로 활성화된다.
정책서 8-11절 (1개 이상 선택 필수).
◼ 선택 전: 흰 배경 + 회색 테두리 ◼ 선택 후: 연보라 배경 + 파란 테두리 + 체크 표시
4
기타 항목 선택 시 텍스트 입력 필드 노출
  • "기타 (직접 입력)" 항목을 탭하면 아래에 텍스트 입력 박스가 나타난다.
  • 최대 200자까지 입력할 수 있다.
  • 입력은 선택 사항이다. 텍스트를 입력하지 않아도 [탈퇴하기] 버튼을 탭할 수 있다.
정책서 8-11절 (기타 선택 시 텍스트 입력 필드 노출, 최대 200자).
5
[탈퇴하기] 버튼 (하단 고정) ★
  • 사유를 1개 이상 선택하면 빨간색으로 활성화된다. 선택 전에는 회색 비활성 상태이다.
  • 탭 시 선택한 사유 데이터를 서버로 보내고 계정을 삭제한다.
  • 삭제가 완료되면 세션(로그인 상태)이 자동으로 지워지고 MY05-C(탈퇴 완료) 화면으로 이동한다.
  • 적립된 캐시백은 이 순간 전액 소멸되며 되돌릴 수 없다.
정책서 8-11절 / 10-4절 (탈퇴 처리 + 세션 삭제 + 캐시백 소멸).
◼ 사유 미선택: 회색 비활성 ◼ 1개 이상 선택: 빨간색 활성 ◼ 탭 후: 서버 처리 → MY05-C 이동

화면 주석 — MY05-C (탈퇴 완료)

1
탈퇴 완료 메시지
  • 계정 삭제가 완료되면 이 화면이 나타난다.
  • 이 시점에서 세션(로그인 상태)은 이미 삭제된 상태이다. 즉, 비로그인 상태로 전환된다.
  • GNB의 뒤로가기 버튼은 없다. 탈퇴를 되돌릴 수 없기 때문이다.
정책서 8-11절 / 10-4절 (탈퇴 완료 후 세션 삭제).
2
재가입 제한 안내 문구 및 날짜 표시
  • 탈퇴한 날짜(KST 기준) 기준으로 7일 후부터 같은 이메일로 재가입이 가능하다.
  • 예시: 7월 1일 탈퇴 → 7월 8일 00:00 이후 재가입 가능.
  • 재가입 가능 날짜를 "YYYY년 MM월 DD일 이후" 형식으로 표시한다.
  • 만약 7일 안에 같은 이메일로 가입을 시도하면, 로그인 화면에서 "회원탈퇴 후 7일간 재가입이 제한됩니다. OO월 OO일 이후 다시 시도해주세요." 메시지가 나타난다.
정책서 8-11절 (탈퇴일+7일 재가입 제한 + 차단 안내 문구).
◼ 7일 이내 재가입 시도: 로그인 화면에서 재가입 차단 안내 문구 표시
3
[메인으로 이동] 버튼
  • 탭 시 서비스 첫 화면(M01 메인)으로 이동한다.
  • 비로그인 상태로 메인 화면이 열린다.
  • 이 버튼을 탭하지 않아도 일정 시간 후 자동으로 메인으로 이동할 수 있다. (구현 시 결정)
정책서 10-4절 (탈퇴 완료 후 M01 메인으로 이동 + 세션 삭제).
