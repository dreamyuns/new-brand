카드 최저가 — 우하단 단일 표시
  • 확정 디자인: 공급사 행 제거, 최저가만 카드 우하단에 표시.
  • 금액: topRates[0].price (isCheapestRate:true 기준).
  • 세금 포함 최종가. "/N박" 단위 함께 표시 — N은 검색 박수(체크아웃 - 체크인 일수). 2박 검색 시 "/2박", 3박 검색 시 "/3박"으로 자동 변경.
  • 표시 금액도 검색 박수에 맞는 합산 금액으로 계산 (예: 1박 ₩200,000 × 2박 검색 = ₩400,000 표시).
  • 천 단위 구분 쉼표, ₩ 기호 표기.
