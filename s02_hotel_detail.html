S02-B1 — 이미지 API 필드

1

  • 전환 시 우하단 카운터(N/M) 동기화
  • 끝에서 순환 없음 (마지막에서 오른쪽 스와이프 → 멈춤)
2
스와이프 화살표 ‹ › — 이미지 좌우 이동

  • 첫 번째 이미지: ‹ 미표시. 마지막: › 미표시
  • 이미지 1장이면 화살표 없음
3
📷 카운터 겸 갤러리 버튼 — 이미지 우하단
  • 표시: "📷 N / M" (N=현재 번호, M=images[].length)
  • 탭 → S02-C 갤러리 바텀시트 오픈. 이미지 영역 탭과 동일 동작
  • 스와이프 시 N 자동 업데이트. 이미지 1장이면 미표시

  • images[].tag — 카테고리 코드. null → "기타". S02-C 갤러리 탭 분류에 사용

S02-B2 — 숙소요약 API 필드

1
호텔명 — name
  • languageCode=ko_KR 요청 시 한국어명 반환
  • 17px, 최대 2줄. 넘치면 줄임말(…)
2
성급 — starRating
  • (integer 1~5) → ★ N개 (#f59e0b)
  • isSelfRated:true → ★ 대신 "S" 뱃지
  • 0 또는 없으면 미표시
3
평점 — reviews.sentiment 우선, guestRatings.OVERALL fallback
  • 파란 뱃지 숫자: reviews.sentiment에서 쉼표(,) 뒤 숫자 추출
  • 레이블 텍스트: reviews.sentiment에서 쉼표(,) 앞 텍스트 추출
  • 예시: "훌륭함, 8.6" → 뱃지 8.6, 레이블 "훌륭함" / "Fabulous, 8.9" → 뱃지 8.9, 레이블 "Fabulous" (언어 설정 따라 자동 현지화)
  • sentiment가 null이면 → guestRatings.OVERALL 숫자만 파란 뱃지에 표시. 레이블 텍스트 없음 (별도 텍스트 매핑 없음)
  • OVERALL도 없으면 평점 영역 전체 미표시
기준: v0.36 6-7절
4
주소 — address (탭 → 클립보드 복사)
  • 단일 문자열, 1줄 표시. 탭 → 주소 텍스트를 클립보드에 복사
  • 복사 완료 직후 화면 상단에 토스트 팝업 표시: "주소를 복사하였습니다"
  • 없으면 미표시. 섹션 전체 탭 동작 없음 — 주소만 클릭 가능
S02-B3 섹션 상세 핵심 공급사목록 — providers[] + results[]
S02 핵심 섹션. providers[](공급사 메타) + results[](객실 요금)를 providerIndex로 연결. 날짜·인원 바·필터 칩은 이 섹션 내 인라인 배치.
⚙ 공통 정책 — 공급사목록 섹션
  • 정렬: totalRate 오름차순. 클라이언트 정렬 (6-3절)
  • 올마이투어 행: MVP 미노출 (6-3절)
  • 필터 칩: API 재호출 없이 클라이언트 실시간 필터. 복수 선택 AND 조건 (6-3절)
  • 날짜·인원 바: 섹션 상단 인라인. sticky 없음 (6-5·6-6절)
  • 잔여객실: availableRooms ≤ 3일 때만 표시. 4개 이상·null 미표시 (6-3절)
기준: v0.36 6-3·6-5·6-6절
