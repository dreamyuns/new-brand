⚙ 공통 정책 — 이메일 템플릿
  • 조회·편집 대상: 서비스에서 자동 발송되는 이메일 본문. 발신자명·제목·본문 텍스트 편집 가능 (9-2절)
  • 발송 트리거: 각 템플릿은 서비스 이벤트(회원가입·캐시백 상태변경 등) 발생 시 자동 발송. 관리자가 트리거 조건 변경 불가 (9-1절)
  • 변수 삽입: 본문 내 {{변수명}} 형식으로 동적 데이터 삽입. 편집 화면에서 사용 가능한 변수 목록 제공 (9-3절)
  • MVP 범위: 5종 편집 가능. 예약 상태 알림(#6)은 자사 직접 예약 연동 시 추가 (Phase 2)
기준: 어드민 v0.3 9절
편집 안내: 각 템플릿의 [편집] 버튼을 클릭하면 발신자명·제목·본문을 수정할 수 있습니다. 변경 사항은 저장 즉시 다음 발송부터 반영됩니다.
