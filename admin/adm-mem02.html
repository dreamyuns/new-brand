1

  • 이 서비스는 이메일 인증코드로 로그인하기 때문에, 이메일이 계정을 구분하는 유일한 식별자임 → 수정 불가

2
이름·성 수정

  • 선택 항목 — 입력하지 않아도 저장 가능
3
국적 변경
  • [변경] 버튼 클릭 → 국가 선택 팝업(바텀시트) 열림 — 프런트 화면 MY03(계정관리) 패턴 참조
  • 국가 목록은 KAYAK Constants API에서 가져온 국가 목록 기반

  • 숫자와 하이픈(-)만 허용. 선택 항목 — 입력 안 해도 저장 가능
5
마케팅 동의 토글

  • 변경 날짜(이력)는 자사 서버에 기록됨
  • 변경 전 "정말 바꾸시겠습니까?" 확인 창(다이얼로그)을 보여주는 것을 권장

  • 탈퇴 회원도 예약내역 탭 조회 가능 — CS(고객 대응) 목적 (케이스 D에서도 유지)
