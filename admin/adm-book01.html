주석 — ADM-BOOK01

1
통합 검색
  • 예약ID(RES-XXXXX)·회원 이메일·호텔명으로 검색
  • KAYAK Reporting API 응답 내 필터 또는 자사 캐싱 DB 검색
2
상태 필터
  • Waiting = 적립예정 (체크아웃 전)
  • Approved = 지급완료 (KAYAK 확인)
  • Cancelled = 취소·만료
정책서 v0.30 9-4절 — 캐시백 상태값
3
공급사(OTA) 필터
  • KAYAK Reporting API providers 기준 OTA명 목록
  • 동적 목록 (고정 값 없음)
4
Excel 다운로드
  • 현재 필터 조건 그대로 전체 다운로드
  • 컬럼: 예약ID·회원·호텔·도시·OTA·체크인/아웃·예약금액·캐시백·상태
개발 공수 확인 후 결정 (8조 미결 3)
5
비로그인 미귀속 건
  • 딥링크 p= 파라미터 없이 발생한 예약 (비로그인 아웃링크)
  • 캐시백 수익은 자사에 귀속되나 회원 귀속 불가
  • 회원 컬럼에 "미귀속" 표시
정책서 v0.30 9절 — 비로그인 캐시백 귀속 불가
6
요약 통계 계산식
  • 전체 예약(1,203): KAYAK Reporting API TransactionType=Booking 전체 건수 합산
  • 적립예정(287): status=Waiting — 체크아웃 전 미확인 상태
  • 지급완료(891): status=Approved — KAYAK 검증 완료 건
  • 취소·만료(25): status=Cancelled — 취소 또는 만료 건
  • 검증: 1,203 = 287 + 891 + 25 ✓
KAYAK Reporting API transactionStatus 필드 / 정책서 v0.30 9-4절
7
예약금액(상품가) 컬럼
  • KAYAK Reporting API transactionAmount 값 — 실제 고객이 OTA에 결제한 상품 금액
  • 세금·수수료 포함 여부는 OTA마다 다름 — "상품가" 표기로 통일
  • 다중 통화 혼재 가능 (₩·¥·THB·SGD 등 그대로 표시)
어드민 v0.3 5-2절 — 예약금액(상품가) 라벨 확정
