주석 — ADM-DASH (참고용)

1
핵심 지표 카드
  • 자사 회원 DB + KAYAK Reporting API(예약·포인트 데이터)를 합쳐서 계산한 값
  • 화면에 표시되는 숫자가 얼마나 자주 새로 고침될지는 개발팀과 협의 필요 (예: 실시간 / 1시간마다 한 번)
  • "오늘" 기준: 자정(0시)부터 현재 시각까지
  • 포인트 단위는 P(포인트). 1P = 1원 기준
우선순위 낮음 — 핵심 화면 완성 후 구현 (어드민 v0.1 3조)
2
최근 예약 5건
  • 예약내역 전체(ADM-BOOK01) 화면의 데이터를 가져와서 가장 최근 5건만 요약해 보여줌
  • [전체 예약 보기] 버튼을 누르면 ADM-BOOK01 화면으로 이동
3
최근 가입 회원 5명
  • 회원 목록(ADM-MEM01) 화면의 데이터를 가져와서 가장 최근에 가입한 5명만 요약해 보여줌
  • [전체 회원 보기] 버튼을 누르면 ADM-MEM01 화면으로 이동
  • 이름을 입력하지 않은 회원은 이름 칸에 "—"으로 표시
