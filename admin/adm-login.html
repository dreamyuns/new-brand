주석 — ADM-LOGIN

1
아이디 입력 필드
  • 텍스트 입력 필드. placeholder: "관리자 아이디 입력"
  • Enter 키 입력 시 비밀번호 필드로 포커스 이동
정책서 어드민 v0.1 2조 — 아이디+비밀번호 인증
2
비밀번호 입력 필드
  • type="password" (마스킹 처리)
  • Enter 키 입력 시 [로그인] 버튼 동작 트리거
  • 눈 아이콘으로 표시/숨김 토글 — 추후 논의
정책서 어드민 v0.1 2조
3
[로그인] 버튼
  • 아이디·비밀번호 모두 입력 시 활성. 미입력 시 비활성(회색)
  • 성공 시 → 대시보드(ADM-DASH)로 이동
  • 실패 시 → 케이스 B 오류 메시지 표시
프론트: 인증 실패 횟수 카운트 — 5회 초과 시 잠금 여부 추후 확정 (8조 미결 2)
4
로그인 오류 메시지 (케이스 B)
  • 문구: "아이디 또는 비밀번호가 올바르지 않습니다. 다시 확인해 주세요."
  • 아이디·비밀번호 필드 테두리 빨간색 강조
  • 오류 시 비밀번호 필드 초기화
5
세션 만료 안내 (케이스 C)
  • 문구: "세션이 만료되었습니다. 다시 로그인해 주세요."
  • 노란색 경고 배경으로 오류와 구분
  • 세션 없이 어드민 URL 직접 접근 시 이 케이스로 리다이렉트
