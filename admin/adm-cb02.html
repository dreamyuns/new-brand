주석 — ADM-CB02

1
기준 월 선택
  • YYYY-MM 형식 드롭다운. 최근 12개월 제공
  • 기본값: 이번 달 (오늘 기준 현재 월)
  • Approved 기준 집계 → 체크아웃 후 KAYAK 확인 완료 건
2
공급사별 집계 테이블
  • 캐시백 합계: 통화별 원본 표시. 여러 통화 혼재 가능
  • 평균 캐시백율: PERCENTAGE 타입만 계산. FLAT/NONE은 "—"
  • NONE 타입 OTA: 캐시백 없음이나 예약 건수는 집계
정책서 v0.30 9-7절 캐시백 타입 / 11-2절 통화 정책
4
적립률/금액 컬럼 — 타입별 표시 분기
  • PERCENTAGE 타입: 적립률(%) 표시 — 예: 3.2%
  • FLAT 타입: 적립 금액 표시 — 통화별로 줄바꿈하여 표시 (예: SGD 240 / ₩120K)
  • NONE 타입: "—" 표시 — 캐시백 없음
  • 한 행에서 타입이 PERCENTAGE이면 % 값만, FLAT이면 금액만 표시. 두 값을 같이 보여주지 않음
어드민 v0.3 6-4절 — KAYAK 캐시백 타입 구분
3
요약 통계 계산식
  • 총 캐시백(Approved): 해당 월 Approved 건 캐시백 금액 합산. 통화 혼재 시 통화별 분리 표시 권장 (KRW 환산 기준 미결)
  • 총 예약 건수: 해당 월 Booking 트랜잭션 전체 — Waiting + Approved + Cancelled 합산
  • 참여 OTA 수: 해당 월 예약 1건 이상 발생한 OTA 수 (KAYAK Reporting API providers 기준)
  • 적립예정(Waiting): 해당 월 발생 건 중 체크아웃 전 미확인 캐시백 금액 합산
  • 평균 캐시백율: PERCENTAGE 타입 OTA의 (캐시백합계 ÷ 예약금액합계) × 100. FLAT·NONE 타입 제외
통화 혼재 시 KRW 환산 기준 미결 — 개발팀 협의 필요
캐시백율 조정 공급사별 캐시백율 조정은 KAYAK 담당자 양식 제출 방식.
어드민에서 직접 변경 불가. Phase 2에서 조정 이력 조회 기능 논의 예정.
