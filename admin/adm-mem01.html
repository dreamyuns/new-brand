주석 — ADM-MEM01

1
통합 검색 필드
  • 회원 이메일 주소나 회원 ID(예: U-10042)로 원하는 회원을 찾을 수 있음
  • 이름(영문)은 회원이 선택으로 입력하는 항목이라 여기서는 검색 불가 → 이름 확인은 [상세] 버튼을 눌러 MEM02에서 확인
  • 입력 후 Enter 키 또는 [검색] 버튼 클릭으로 검색. 2자 이상 입력 권장
어드민 v0.3 4-1절
2
[탈퇴회원만 보기] 버튼
  • 평상시: 버튼이 회색 테두리로 표시 → 탈퇴하지 않은 활성 회원만 목록에 나옴
  • 버튼 클릭 시: 버튼이 빨간 테두리·글씨로 강조되고, 탈퇴 처리된 회원만 새로 불러옴
  • 탈퇴 회원은 고객 문의(CS) 대응 목적으로 조회 허용 → 예약내역·포인트 탭도 확인 가능
  • 다시 클릭하면 활성 회원 목록으로 돌아옴
어드민 v0.3 4-1절 · 4-3절 — 탈퇴 회원 CS 조회 정책
3
가입기간 필터
  • 특정 기간 사이에 가입한 회원만 보고 싶을 때 사용. 날짜 직접 입력 또는 달력으로 선택
  • 시작일만 입력하면 그 날 이후 가입한 회원 전체를 조회
  • [탈퇴회원만 보기] 모드에서는 가입기간 필터가 탈퇴기간 필터로 자동 전환됨
4
Excel 다운로드
  • 지금 화면에 적용된 검색 조건과 관계없이 전체 회원 데이터를 엑셀로 내려받음
  • 엑셀 파일 안에 시트가 2개로 나뉨:
    • 시트1 "활성회원" — 회원ID · 이메일 · 상태 · 가입일 · 예약 수
    • 시트2 "탈퇴회원" — 회원ID · 이메일 · 탈퇴일 · 가입일 · 예약 수
  • 이름·전화번호 등 개인정보가 포함되므로 다운로드 기록을 남기는 것을 권장
시트 분리 방식 개발팀 협의 필요 (8조 미결 3)
5
예약 수 컬럼
  • 해당 회원에게 포인트 지급이 확정된(Approved) 예약 건수만 세어 표시
  • 아직 확정 안 된(Active) 건이나 취소된(Cancelled) 건은 포함하지 않음
  • 로그인 없이 이용한 경우는 회원에 귀속되지 않아 이 숫자에 들어가지 않음
어드민 v0.3 4-1절
7
누적 포인트(P) 컬럼
  • 해당 회원에게 지급완료(Approved)된 포인트를 모두 더한 값. 단위는 P(포인트), 1P = 1원
  • 아직 확정 안 된(Active) 건이나 취소된(Cancelled) 건은 합산에서 제외
  • NONE 타입(포인트 없는 예약) 건도 제외
  • 포인트가 한 번도 없었거나 0인 경우 → "—" 표시
어드민 v0.3 6-4절 — 포인트 단위 P, 1P=1원
6
[상세] 버튼
  • 클릭하면 ADM-MEM02 회원 상세 화면으로 이동 — 해당 회원의 정보·예약·포인트 내역을 볼 수 있음
  • 탈퇴 회원의 [상세(읽기전용)] 버튼도 MEM02로 이동 — 정보 수정은 안 되고 조회만 가능한 상태(케이스 D)로 열림
이름·마지막 로그인 컬럼 미제공 이름(영문)은 선택 입력 항목이라 없는 회원이 많아 목록에는 표시하지 않음.
이름과 마지막 로그인 일시는 상세(MEM02)에서 확인 가능.
