주석 — ADM-MEM01

1
통합 검색 필드
  • 이메일·회원ID 대상 검색
  • 이름(영문)은 회원이 선택 입력하는 항목이라 목록에서 검색 불가 — 이름 확인은 MEM02 상세에서
  • 입력 후 Enter 또는 [검색] 버튼 클릭
  • 검색어 2자 이상 입력 시 검색 활성 권장
어드민 v0.3 4-1절
2
[탈퇴회원만 보기] 버튼
  • 기본 상태: 활성 회원만 표시 (버튼 비활성 스타일)
  • 버튼 클릭 시: 버튼이 빨간색 강조 스타일로 바뀌며, 목록이 탈퇴 회원 전용으로 새로 불러와짐
  • 탈퇴 회원 조회는 CS 대응 목적 — 예약내역·캐시백 탭 조회 가능
  • 버튼을 다시 클릭하면 활성 회원 목록으로 돌아옴
어드민 v0.3 4-1절 · 4-3절 — 탈퇴 회원 CS 조회 정책
3
가입기간 필터
  • 날짜 직접 입력 또는 캘린더로 선택
  • 시작일만 입력 시 해당일 이후 전체 조회
  • 탈퇴회원 모드에서는 가입기간 대신 탈퇴기간 필터로 변경됨
4
Excel 다운로드
  • 검색 조건과 관계없이 전체 회원 데이터를 다운로드
  • 파일 내 시트 2개로 분리됨:
    • 시트1 "활성회원" — 회원ID·이메일·상태·가입일·예약수
    • 시트2 "탈퇴회원" — 회원ID·이메일·탈퇴일·가입일·예약수
  • 개인정보 포함 — 다운로드 로그 기록 권장
시트 분리 방식 개발팀 협의 필요 (8조 미결 3)
5
예약 수 컬럼
  • Approved(지급완료) 기준 집계 — Waiting·Cancelled 미포함
  • KAYAK Reporting API 해당 회원 labels 기준 조회
  • 비로그인 미귀속 건 제외 (회원ID 매핑된 건만 집계)
어드민 v0.3 4-1절
7
누적 캐시백(KRW) 컬럼
  • 해당 회원의 Approved(지급완료) 캐시백만 합산 — Waiting·Cancelled 제외
  • KAYAK API의 NONE 타입 건은 합산에서 제외
  • KRW 원화 기준 단일 통화 — 환산 없이 직접 합산
  • 캐시백 0원 / 적립 이력 없음 → "—" 표시
어드민 v0.3 6-4절
6
[상세] 버튼
  • 클릭 시 ADM-MEM02 회원 상세 페이지로 이동
  • 탈퇴 회원의 [상세(읽기전용)] 버튼도 MEM02로 이동 — 케이스 D(읽기전용)로 표시됨
  • URL: /admin/members/:userId
이름·마지막 로그인 컬럼 미제공 이름(영문)은 선택 입력 항목이라 목록에서 의미가 낮아 상세(MEM02)에서만 표시.
마지막 로그인도 상세(MEM02)에서 확인 가능.
