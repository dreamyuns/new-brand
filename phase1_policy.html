제1조

목적 및 MVP 범위

1-1·1-2서비스 목적 · MVP 핵심 가치

KAYAK API를 통한 다중 OTA 호텔 가격 비교 제공. 로그인 회원 기반 캐시백 적립 및 월 정산 후 고객 직접 지급.

  • 수익 구조: KAYAK CPA 100% 의존 (자사 OTA 수익 없음 — MVP 한정)
  • 수익 배분: 자사 60% : KAYAK 40%
  • 언어: 한국어(기본) + 8개 언어 (영어·일본어·스페인어·인도네시아어·베트남어·중국어번체)
  • 통화: 원화(기본) + 카약 확정 20개 고정 목록

제3조

MVP 화면 플로우

기본 플로우: M01(메인) → S01(검색결과) → S02(숙소상세, 새탭) → OUT01(아웃링크 이동중) → 타 OTA

지도 진입: S01 [지도] 탭 → S01-MAP(?layout=map) → 핀 선택 → S02 새탭

GNB 패턴 원칙 (v0.37 수정):

  • M01 (메인): 좌측 [로그인] 버튼(비로그인) / 아바타(로그인) + 중앙 브랜드 로고 + 우측 캐시백 버튼. [☰ 전체메뉴] 제거 (v0.37)
  • S01 (검색결과): ← 뒤로가기 + 검색조건 요약(중앙) + 필터아이콘(우측). 로고 패턴 폐기 (v0.37 확정)
  • 세부탐색형 (S02·AUTH·OUT01): ← 뒤로가기 + [로그인] + [공유하기]
  • 마이페이지 하위 (MY01~MY04): ← 뒤로가기만 (로그인·전체메뉴 없음)

S01→S02 이동: window.open() 새탭. S02 GNB ← = window.close(), 실패 시 M01 fallback

4-1자동완성 개요
  • API: GET /api/affiliate/autocomplete/v1/hotels
  • 최대 결과: 6개
  • 트리거: 완전한 음절 1자 이상. 자음·모음 단독(ㅅ, ㅓ 등)은 트리거 안 함
  • placeType: airport / city / country / hotel / trainstation / region / touristregion / neighborhood / landmark / nationalpark / island
  • ✅ Autocomplete API 화이트리스트 등록 완료 (2026-06-10)
