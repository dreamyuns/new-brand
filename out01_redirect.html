  • 이동 방식 *(v0.32 확정)*: S02 [이동→] 탭 → window.open()으로 OUT01 새탭 오픈 → OUT01에서 2초 노출 후 window.location.href = bookUri로 공급사 페이지 직접 이동. 공급사 별도 새탭 미사용 (7-1절)
  • GNB ← 동작 *(v0.32 확정)*: window.close() → OUT01 탭 닫힘. 실패 시 진입 화면 fallback — S01 진입→S01, S02 진입→S02 (7-4절)
  • userTrackId: 로그인 여부 무관 항상 포함 (7-2절)
  • 캐시백 안내 배너: 로그인 회원 + cashbackType ≠ NONE 시만 표시. 비로그인 또는 cashbackType=NONE=미표시 (7-4절)
  • 캐시백 금액: 로그인 회원이고 cashback.type = PERCENTAGE이면 💸 캐시백 최대 N%, FLAT이면 💸 캐시백 최대 ₩N 표시 (7-4절)
