§0

용어 정의

BCP-47
IETF가 정의한 언어 태그 표준. 하이픈(-)으로 언어-지역을 구분. UI·콘텐츠 기준으로 사용. 예: ko-KR, en-US, zh-TW
ISO 639
ISO가 정의한 언어 코드 국제 표준. KAYAK·Expedia API는 언더스코어(_) 형식을 요구. 예: ko_KR, en_US, zh_TW
ICP 라이선스
중국 정부(공업정보화부)가 인터넷 서비스 제공자에게 요구하는 라이선스. 중국 본토에서 서비스하려면 반드시 취득 필요. 취득 불가 시 zh-CN(간체) 서비스 불가 → D2 제외 근거.
Ketch
쿠키·개인정보 동의를 관리하는 CMP(Consent Management Platform) 솔루션. API 기반으로 국가별 동의 정책을 코드로 설정 가능. AWS 엣지와 통합 정합성이 높아 선정.
CMP
Consent Management Platform. 사용자의 쿠키·개인정보 수집 동의를 받고 관리하는 솔루션. GDPR·PIPA 등 규정 준수를 위해 필수. 국가별로 다른 동의 방식(옵트인/옵트아웃)을 처리.
Lambda@Edge
AWS CloudFront(CDN) 엣지 서버에서 실행되는 서버리스 함수. 사용자 요청이 오리진 서버에 도달하기 전, 엣지에서 국가 감지·로케일 설정 등을 처리 가능. 지연 5ms 미만.
Geo 감지
사용자의 IP 주소를 기반으로 접속 국가를 자동 감지하는 기술. CloudFront가 CloudFront-Viewer-Country 헤더로 국가 코드를 전달. 언어·통화 자동 설정에 사용.
링구아 프랑카
모국어가 다른 사람들 사이의 공통 언어. 이 정책에서는 영어(EN)가 링구아 프랑카 역할을 담당 — 12개 통화권(USD·GBP·CAD·AUD 등)을 영어 UI 하나로 커버.
GDPR
EU 일반 개인정보 보호 규정. 유럽 사용자 개인정보 처리 기준. 동의 없이 마케팅·분석 쿠키 수집 불가. 위반 시 전 세계 매출의 최대 4% 과징금.
PIPA
한국 개인정보 보호법 (Personal Information Protection Act). 국내 서비스에 적용. GDPR 대비 간소 옵트인 방식 허용 — 수락 버튼 표시 + 거부 가능 구조.
APPI
일본 개인정보 보호법 (Act on Protection of Personal Information) + 전기통신사업법. 고지형(옵트아웃) 방식 — 쿠키 사용 고지 후 사용자가 거부하지 않으면 수락으로 간주.
CCPA / CPRA
미국 캘리포니아 소비자 프라이버시법 / 프라이버시 권리법. "내 개인정보 판매 금지(Do Not Sell)" 링크를 필수로 제공해야 함.
LGPD
브라질 일반 개인정보 보호법 (Lei Geral de Proteção de Dados). GDPR과 유사한 강제 옵트인 구조. 수락·거부를 동등하게 제공해야 함.
Typeahead
검색창 키 입력 시 실시간으로 후보 목록을 보여주는 자동완성 기능. Expedia API 명칭. KAYAK의 Autocomplete API와 동일한 기능. Lodging(숙소) API와 별도로 동작.
Lodging API
Expedia의 숙소 데이터·가격 API. 지원 언어·통화가 Typeahead(자동완성) API보다 제한적. 1차 지원 언어 7종은 모두 Lodging 지원 범위 내에 있음.
폴백 (Fallback)
요청한 조건(언어·통화)이 지원되지 않을 때 기본값으로 대체하는 처리 방식. 이 정책에서는 미지원 언어 → en_US, 미지원 통화 → USD, 국가 미식별 → ko-KR/KRW로 폴백.

⚠️ 중국어 주의 사항
zh-CN(간체/본토)·zh-TW(번체/대만)·zh-HK(광둥어/홍콩) — 별개 코드, 혼용 금지.

7개 언어만으로 20개 통화 표시를 충족. USD·GBP·CAD·AUD·NZD·SGD·INR·ZAR·PHP·HKD·MYR·EUR 등 12개 통화권을 영어 UI로 커버.
Expedia 호환성 Lodging 지원 범위 확인

Expedia Lodging API는 전체 언어를 지원하지 않음. 1차 지원 UI 언어 7종은 모두 Lodging 지원 범위 내.

폴백 규칙: 7종 범위 외 언어로 Expedia API 호출 시 → en_US로 폴백 처리

  • Lodging 미지원 언어 (Typeahead 전용): en-AU/CA/IN, es-AR/US, fr-BE, de-AT, nl-BE 등 → 자동완성만 지원, Lodging 호출 불가
  • Lodging 미지원 통화 (언어는 지원): UAH(우크라이나)·HUF(헝가리)·ISK(아이슬란드) → 통화 매핑 불가, USD 폴백
