화면 주석 — MY01-A (비로그인)

진입
GNB [☰ 전체메뉴] 탭 → 마이페이지 전체화면
  • M01·S01·S02 어느 화면에서든 GNB [☰] 탭 시 마이페이지 전체화면으로 직접 이동. 별도 드롭다운 없음.
  • 마이페이지 하위 화면(MY01~MY03) GNB에는 뒤로가기(←) 버튼만 표시. 로그인 버튼·전체메뉴 버튼 미표시.
정책서 5-1절·6-1절·10-1절.
뒤
GNB [←] 뒤로가기 — 마이페이지 닫기
  • 탭 시 마이페이지를 닫고 직전 화면(S01 검색결과·S02 숙소상세 등)으로 복귀. MY01~MY03 공통 GNB 패턴.
정책서 10-1절. MY01~MY03 공통 패턴.
1
로그인 유도 영역 — 상단 고정
  • 비로그인 상태 전용 영역. 캐시백 혜택 헤드카피와 설명 문구를 노출해 로그인 동기 부여.
  • 회색 배경(#f8f8f6)으로 메뉴 목록과 시각적 구분. 하단에 [회원가입/로그인] 버튼 배치.
정책서 10-1절 (비로그인 상단 로그인 유도 영역).
2
[회원가입/로그인] 버튼 ★
  • 탭 시 → 로그인/회원가입 통합 바텀시트(auth_login.html — AUTH-A) 팝업.
  • 버튼 텍스트는 "회원가입/로그인" 단일 표기 (로그인·회원가입 분리 없음).
정책서 8-1절. 로그인·회원가입 통합 버튼 (v0.13 확정).
3
국가·지역 메뉴
  • 탭 시 국가·지역 선택 바텀시트(MY01-C)가 슬라이드업으로 올라와요. 우측에 현재 선택된 국가·지역명을 미리보기로 표시해요 (예: 대한민국).
  • 국가·지역을 선택하면 도메인이 해당 지역으로 변경되고, 언어도 함께 바뀌어요. 통화 설정은 따로 적용되어 자동 변경되지 않아요.
정책서 11-3절·11-4절. 도메인 변경 = 언어+시장 동시 변경. 통화 독립 유지.
4
통화 변환 메뉴
  • 탭 시 통화 선택 바텀시트(MY01-D) 슬라이드업. 현재 선택된 통화 코드를 우측에 미리보기 표시(예: KRW).
  • 통화 변경은 검색결과·숙소상세 등 모든 가격 표시에 즉시 반영. API 호출의 currencyCode 파라미터 변경.
통화 목록: KAYAK Constants API 전체 통화 동적 사용 (고정 목록 없음). 기본값 KRW. 정책서 11-2절 v0.18.
5
약관 메뉴
  • 탭 시 약관 전문 바텀시트 팝업. 이용약관·개인정보처리방침 내용 포함.
  • 비로그인·로그인 공통 메뉴. MVP에서는 내용 표시 전용 (별도 동의 액션 없음).
정책서 10-1절 공통 메뉴.

회원 이름
  • 계정관리(MY03)에서 입력한 이름(First Name) + 성(Last Name) 영문 조합으로 표시 (예: Sungkyoon Yoon).
  • 이름·성 모두 미입력 시: 이메일 주소의 @ 앞 부분을 대체 표시.

회원 이메일

정책서 10-3절 (이메일 읽기전용, 변경 불가).

정책서 10-1절·10-2절. 로그인 전용 메뉴.

계정관리 메뉴

  • 수정 가능 항목: 이름·성(영문), 국적, 전화번호, 마케팅 수신 동의. 이메일은 읽기전용.
  • 로그인 전용 메뉴. 비로그인 상태에서는 미표시.
정책서 10-1절·10-3절. 로그인 전용 메뉴.

화면 주석 — MY01-C (국가·지역 선택)

진입
국가·지역 메뉴 탭 → 바텀시트 슬라이드업
  • MY01-A(비로그인) 또는 MY01-B(로그인) 화면에서 국가·지역 메뉴를 탭하면 이 바텀시트가 아래에서 올라와요. 로그인 여부 구분 없이 동일하게 나타나요.
  • 어두운 배경(딤 영역)을 탭하거나 [✕] 버튼을 탭하면 바텀시트가 닫혀요. 아무것도 선택하지 않고 닫으면 기존 국가·지역이 그대로 유지돼요.
정책서 v0.35 10-1절·11-3절.
1
바텀시트 타이틀
  • 상단에 "국가·지역 선택" 텍스트가 고정으로 표시돼요. 우측 [✕] 버튼을 탭하면 아무 것도 바뀌지 않고 바텀시트가 닫혀요.
2
국가·지역 항목 선택 동작
  • 항목 탭 → 해당 도메인으로 변경 → 바텀시트 닫힘 → 현재 화면 유지 (다른 페이지로 이동하지 않음).
  • 도메인이 바뀌면 언어와 시장(서비스 제공 지역)이 함께 변경돼요. 예: 🇺🇸 United States 탭 시 영어 UI로 전환.
  • 통화 설정은 자동으로 바뀌지 않아요. 통화를 바꾸려면 MY01-D(통화 선택)에서 따로 변경해야 해요.
  • 현재 선택 중인 국가·지역 항목에는 체크마크(✓)가 표시돼요. 기본값: 대한민국(한국어).
정책서 v0.35 11-3절. 도메인 변경 = 언어+시장 동시 변경. 통화 독립 유지(11-4절).
그루핑 없는 플랫 목록. 총 19개 항목: 대한민국·日本·US·UK·AU·CA·SG·NZ·IN·ZA·PH·HK·MY·Europe·MX·ES·ID·VN·TW. 선택 중인 항목에만 ✓ 표시.

화면 주석 — MY01-D (통화 선택)

진입
통화 메뉴 탭 → 바텀시트 슬라이드업
  • 비로그인·로그인 공통. 언어 선택(MY01-C)과 동일 바텀시트 구조 사용.
  • 딤 영역 탭 또는 [✕] 탭 시 바텀시트 닫힘. 선택 없이 닫으면 기존 통화 유지.

1
바텀시트 타이틀

2
현재 선택 통화 — ✓ 표시
  • 현재 적용 중인 통화 항목에 인디고 체크마크(✓)와 텍스트 색상 강조 표시.

  • 기본값: 원화(KRW).
기본값: KRW. 전체 목록 KAYAK Constants API 동적 로딩 (앱 초기화 시 캐싱). 정책서 11-2절 v0.18.
API 파라미터: currencyCode = Constants API 제공 값 (예: KRW·USD·JPY·CNY·SGD·EUR·MYR 등)
3
통화 목록 — 동적 로딩 ★
  • 통화 선택 목록은 KAYAK Constants API(/api/4.0/constants-mapping) 전체 통화를 앱 초기화 시 캐싱해 사용.
  • 고정 4개(KRW·USD·JPY·CNY) 방식에서 전체 통화 동적 방식으로 변경 (v0.18).
  • 영어권 국가 통화(싱가포르 SGD·말레이시아 MYR 등) 포함.
  • 와이어프레임에서는 대표 7개 샘플 표시 — 실제 목록은 Constants API 응답 전체.
정책서 11-2절 v0.18 (2026-06-12 대표님 방향 확정).
